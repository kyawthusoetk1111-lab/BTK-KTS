# **App Name**: QuizCraft Pro

## Core Features:

- Quiz Creation: Enable teachers to create quizzes with names and descriptions.
- Section Management: Allow the adding, editing, and deleting of quiz sections.
- Question Management: Support the adding, editing, and deleting of questions within sections, supporting multiple question types: multiple-choice, short-answer, essay, matching, dropdowns, and passage-based questions.
- Dynamic Forms: Create a dynamic form interface allowing real-time editing, addition, and removal of quiz elements (sections, questions, options).
- AI auto-suggests points: Suggest adequate scores for different answers or subproblems of an essay, in order to ensure grading integrity.
- AI correct/incorrect designation tool: Using an LLM to evaluate a database of answers of closed-ended (such as multiple choice or numerical entry) problems in order to automatically determine and suggest the ground truth for the question
- Database integration: Cloud based, persistent database to store, update and retrieve tests.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), a regal and intellectual hue, suitable for an educational application.
- Background color: Very light indigo (#F0F8FF), provides a calm backdrop.
- Accent color: Violet (#8F00FF), adding visual interest for interactive elements.
- Body text: 'Inter', a grotesque sans-serif font known for its clean and modern appearance, making it highly readable and accessible.
- Headline Font: 'Space Grotesk', sans-serif, a contemporary font choice, ensuring headlines are clear, engaging, and align with a modern design aesthetic.
- Use minimalist icons for quiz, section, and question actions to provide visual clarity and intuitive navigation.
- Implement subtle transitions and animations for adding/removing questions or sections to enhance user experience without distraction.
